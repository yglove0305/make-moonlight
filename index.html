안녕하세요. 우리 달빛 만들기는 더 좋은 디자인과 기능을 위해 업데이트를 진행하고 있습니다. 이 화면이 뜨는 이유는 (깃허브 로고 사진과) Github 에서 보안 확인 절차를 진행하고 있어서 현재는 이용이 불가능 합니다. 업데이트가 성공이 됄 시간은 한국 시간 기준으로 오늘 오후 5시 또는 6시 입니다. 그리고 업데이트가 돼었는데 작동이 되지 않으면 최신 브라우저 로 업데이트 를 해주시길 바랍니다. 양해 부탁드립니다.

업데이트가 완료되었는지 어떻게 확인하나요?

페이지를 새로고침하거나 브라우저 캐시를 비운 뒤 다시 접속해 보세요. 여전히 이용이 불가능하면 브라우저를 최신 버전으로 업데이트한 후 재시도해 주세요.

브라우저는 어떤 것을 권장하나요?

네. 보안 확인 절차는 데이터 보호를 강화하기 위한 과정이며, 사용자 데이터는 안전하게 관리됩니다.

브라우저 업데이트 가이드

  • Chrome: 설정 → 정보 → Chrome 정보에서 업데이트를 확인하고 적용하세요.
  • Edge: 설정 → 정보 → Microsoft Edge 정보에서 업데이트를 확인하세요.
  • Safari (macOS): 시스템 설정 → 일반 → 소프트웨어 업데이트에서 최신 버전으로 업데이트하세요.
  • Firefox: 도움말 → Firefox 정보에서 업데이트를 확인하고 재시작하세요.
